# BB 몬스터 패턴 에디터

PPT로만 표현하던 몬스터 패턴을 실제 시간에서 어떻게 동작하는지 보여주는 에디터.

## 실행 방법

   ```
   https://w8err.github.io/Pattern-Editor/
   ```

위 주소를 크롬/엣지로 연다.

## 웹 팀원용: 배포 데이터 자동 로드 (읽기 + 로컬 편집)

웹사이트만 쓰는 팀원은 **폴더 열기 없이** 접속하자마자 `data/`가 트리에 자동으로 뜬다(보기·플레이테스트).
값 수정도 가능하지만 수정사항은 공유되지 않는다.
`↺ 로컬 편집 초기화`로 배포 데이터로 되돌릴 수 있다. 크롬/엣지가 아니어도 보기는 된다.

> **소유자 주의**: 이 자동 로드는 `data/manifest.json`(파일 목록)에 의존한다. `data/`에 파일을 **추가·삭제·이름변경**하면 배포 전 반드시 재생성:
> ```powershell
> node tools/gen-manifest.mjs
> ```
> 그리고 `manifest.json`을 함께 커밋·push. (생성한 데이터 `.json`도 같이 커밋)

## 사용

1. **📂 폴더 열기** → 데이터를 저장할 실제 폴더 선택 (clone한 repo의 `data/`)
2. 사이드 툴바: `📁＋`(폴더) · `👑＋`(보스) · `👾＋`(몬스터) · `🗑`(삭제)
3. 파일 선택 → 우측 인스펙터에서 편집 → **💾 저장** (엔티티 1개 = `.json` 1개)
4. 다음 접속 시 상단 **↻ 다시 열기**로 같은 폴더 재연결
